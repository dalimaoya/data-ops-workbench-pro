## 1. 设计原则

| 原则 | 说明 |
|---|---|
| 平台库与业务库分离 | 平台库只存配置/日志/任务/版本/元信息，业务库存结果表数据 |
| 主键统一 | 所有表使用 `id: bigint` |
| 审计字段统一 | `created_by / created_at / updated_by / updated_at / is_deleted` |
| 状态枚举化 | 不存自由文本状态 |
| 不过度设计 | 首版单租户、单系统，不引入复杂租户模型 |
| 字符集 | UTF8MB4 |
| 时间字段 | 统一 `datetime` |
| 逻辑删除 | `is_deleted`，0=未删除，1=已删除 |

## 4. 表关系

```
datasource_config (1) → (N) table_config (1) → (N) field_config
table_config (1) → (N) template_export_log
table_config (1) → (N) import_task_log (1) → (N) writeback_log
table_config (1) → (N) table_backup_version
system_operation_log：通用审计表，可关联任意模块
```

## 5. 编码规则

| 类型 | 格式 | 示例 |
|---|---|---|
| 数据源编码 | DS_YYYYMMDD_序号 | DS_20260319_001 |
| 纳管表编码 | TB_YYYYMMDD_序号 | TB_20260319_001 |
| 导出批次号 | EXP_yyyyMMddHHmmss_随机码 | EXP_20260319153000_A3F2 |
| 导入批次号 | IMP_yyyyMMddHHmmss_随机码 | IMP_20260319153100_B7D1 |
| 回写批次号 | WB_yyyyMMddHHmmss_随机码 | WB_20260319153200_C9E4 |
| 备份版本号 | BK_yyyyMMddHHmmss_随机码 | BK_20260319153200_D2F6 |

## 7. 风险提醒

1. 不把业务整表数据备份到平台库，备份保留在业务库
2. 不上复杂租户模型
3. 不首版做逐字段审计
4. 字段含义由管理员确认，不依赖自动猜测
