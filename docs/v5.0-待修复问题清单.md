# v5.0 待修复问题清单（编译版）

## P0 阻塞

### BUG-1: launcher.log 文件不存在
- 日志路径计算错误，`frozen` 模式下 `os.path.dirname(sys.executable)` 指向 `server/app/`，往上一级是 `server/`，再上才是 release root
- 修复：日志目录应该是 `release_root/logs/`

### BUG-2: pywebview 窗口不显示
- webview 加载成功（HAS_WEBVIEW=True），但窗口没弹出
- 可能原因：server 启动后健康检查没通过，所以没调 `_open_main_window`
- 需要在 `_start_server`、`_health_check`、`_open_main_window` 中加日志

### BUG-3: 插件中心显示"没有找到匹配的插件"
- 测试环境正常（19个插件），打包版异常
- 可能原因：`/api/plugins/all` 接口在打包版返回了空数据或错误
- 需要检查打包版的 plugin_loader 是否能正确扫描插件目录

### BUG-4: 微信扫码登录无反应/10052 错误
- 扫码成功后页面无跳转
- 10052 = OAuth state 已被消费（重复扫码）
- 可能原因：回调 redirect_uri 在打包版配置不同，或 AuthCallback 页面加载失败

## P1

### BUG-5: 验证码字符偏小、靠上
- Linux 测试环境正常，Windows 打包版字符偏小偏上
- 可能原因：打包版缺少字体文件，fallback 到默认小字体
- 检查 captcha 生成代码的字体加载逻辑
