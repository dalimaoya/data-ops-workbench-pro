## 4. 各页面核心设计

### 4.1 登录页

**目标：** 完成用户登录，进入系统首页。

**字段：**
- 用户名（文本输入，必填）
- 密码（密码输入，必填）
- 记住登录状态（复选框，可选）

**按钮：** 登录

**状态提示：** 用户名为空 / 密码为空 / 用户不存在 / 密码错误 / 登录成功跳转首页

---

### 4.2 首页/工作台总览

**目标：** 系统入口总览，快速感知当前状态。

**4 个卡片区：**

| 区域 | 内容 |
|---|---|
| 基础统计 | 数据源数量、已纳管表数量、今日导出/导入/回写次数 |
| 最近操作 | 最近 10 条：操作时间、操作人、操作类型、目标表、状态 |
| 待处理提醒 | 表结构变化提醒、导入失败提醒、配置失效提醒 |
| 快捷入口 | 新建数据源、配置表、数据维护、查看日志、版本回退 |

---

### 4.3 数据源管理列表页

**筛选区：** 数据源名称、数据库类型、启用状态

**表格字段：** 数据源名称、数据库类型、主机地址、端口、库名、连接状态、最近测试时间、启用状态、备注

**操作：** 查看 / 编辑 / 测试连接 / 禁用启用 / 删除

**顶部按钮：** 新建数据源

---

### 4.4 新建/编辑数据源页

**表单字段：**
- 数据源名称（必填）
- 数据库类型（必填，MySQL/PostgreSQL/SQL Server）
- 主机地址（必填）
- 端口（必填）
- 数据库名称/默认 schema
- 用户名（必填）
- 密码（必填）
- 编码方式（默认 UTF-8）
- 连接超时秒数
- 启用状态
- 备注

**按钮：** 测试连接 / 保存 / 保存并继续配置表 / 取消

---

### 4.5 纳管表列表页

**筛选区：** 数据源、库/schema、表名、配置状态

**表格字段：** 数据源名称、库/schema、表名、表别名、字段数、主键字段、配置版本、最近结构检查时间、结构状态、启用状态

**操作：** 查看配置 / 配置字段 / 进入维护 / 检查表结构 / 禁用启用

**顶部按钮：** 新增纳管表 / 批量检查结构

---

### 4.6 表配置详情页

**区域 1 - 表基础信息：** 数据源、库/schema、表名、表别名、表说明、当前配置版本、最近同步时间、当前结构状态

**区域 2 - 维护规则：**
- 是否启用维护
- 是否允许导出当前数据 / 全量数据
- 是否允许上传回写
- 是否允许新增记录（首版默认关闭）
- 是否允许删除记录（首版默认关闭）
- 备份保留数量（默认 3）
- 是否强校验模板版本 / 字段顺序

**区域 3 - 主键与唯一标识：** 主键字段选择、唯一识别字段组合

**区域 4 - 样例数据预览：** 展示前 3~10 条样例数据

**按钮：** 保存配置 / 保存并进入字段配置 / 重新拉取表结构 / 取消

---

### 4.7 字段配置页

**表格字段：** 排序号、字段名、字段中文名、数据类型、示例值、是否展示、是否可编辑、是否必填、是否主键、是否系统字段、是否参与导出、是否参与导入、文本长度限制、枚举值配置、备注

**交互：** 支持逐字段编辑、批量设置"仅展示/可编辑"、恢复默认字段名

**默认策略：**
- 所有字段默认展示、默认参与导出
- 主键字段默认不可编辑
- 系统时间字段默认不可编辑

**按钮：** 自动生成默认配置 / 保存 / 从数据库重新同步字段 / 返回表配置

---

### 4.8 数据维护列表页

**目标：** 普通用户核心入口，选择要维护的表。

**表格字段：** 表别名、数据源、库/schema、表名、最近更新人、最近更新时间、今日回写次数、当前结构状态

**操作：** 进入维护 / 导出模板 / 查看日志

---

### 4.9 数据浏览与筛选页

**顶部信息栏：** 当前数据源、当前表名、当前配置版本、当前结构状态

**筛选区：** 通用关键字搜索、按字段筛选（多条件）、重置、查询

**数据操作区按钮：** 导出当前数据模板 / 导出全量数据模板 / 上传修订模板 / 刷新数据

**表格：** 动态字段，统一文本展示，分页，横向滚动，主键固定左侧

**分页：** 默认 20 条/页，支持 20/50/100 切换，服务端分页

---

### 4.10 模板导出确认弹窗

**字段：** 导出类型（当前筛选/全量）、数据行数预估、当前配置版本、导出备注

**按钮：** 确认导出 / 取消

**说明：** 仅支持使用平台导出模板回传，请勿修改模板中字段顺序与隐藏信息

---

### 4.11 模板导入页

**上传区：** 拖拽上传 / 点击选择，展示格式说明

**文件信息区：** 文件名、上传时间、识别出的数据源/表名/配置版本、数据行数、模板有效性初检结果

**按钮：** 开始校验 / 清空文件 / 返回数据页

---

### 4.12 导入校验结果页

**汇总卡片：** 总记录数、可通过数、失败数、警告数、拟修改数

**明细表格：** 行号、主键值、校验状态、错误原因、警告原因、是否允许继续

**按钮：** 查看差异预览 / 导出错误明细 / 取消本次导入 / 返回重新上传

---

### 4.13 差异预览页

**顶部信息：** 当前表、批次号、配置版本、导入人、导入时间

**汇总卡片：** 拟更新记录数、涉及字段数、拟失败记录数、备份策略说明

**差异表格：** 行号、主键值、字段名、原值、新值、差异状态

**按钮：** 确认写入 / 返回校验结果 / 取消本次操作

**关键规则：** 仅当无失败项时允许确认写入

---

### 4.14 写入结果反馈（弹窗/页面）

**内容：** 写入是否成功、成功/失败记录数、备份版本号、写入时间、操作人

**按钮：** 查看回写日志 / 返回数据维护 / 查看版本回退

---

### 4.15 日志中心页

**页签：** 系统操作日志 / 模板导出日志 / 模板导入日志 / 回写日志

**通用筛选：** 数据源、表名、操作类型、操作人、时间范围、状态

**回写日志字段：** 批次号、数据源、表名、操作人、文件名、成功数、失败数、备份版本号、操作时间、状态

---

### 4.16 版本回退页

**筛选区：** 数据源、表名、版本号、时间范围、操作人

**表格字段：** 版本号、数据源、表名、备份时间、备份触发批次、操作人、记录数、是否可回退

**操作：** 查看详情 / 执行回退

**回退确认弹窗：** 目标表、版本号、备份时间、原操作批次、风险提示 → 确认回退 / 取消

## 5. 页面跳转关系

### 管理员主链路
```
登录 → 首页 → 数据源管理 → 新建数据源 → 纳管表列表 → 表配置详情 → 字段配置 → 数据维护列表
```

### 普通用户主链路
```
登录 → 首页 → 数据维护列表 → 数据浏览 → 导出模板 → 本地修改 → 模板导入 → 校验结果 → 差异预览 → 写入反馈 → 日志中心
```

### 异常链路
```
校验失败 → 导出错误明细 → 本地修正 → 重新上传
表结构变化 → 阻断写入 → 管理员进入表配置重新确认
```

## 6. 通用设计规范

| 项目 | 规范 |
|---|---|
| 时间格式 | `YYYY-MM-DD HH:mm:ss` |
| 动态表字段显示 | 统一文本、主键靠左固定、可编辑字段加标识、系统字段置灰 |
| 分页 | 默认 20 条，支持 20/50/100，大表服务端分页 |
| 错误提示 | 必须说清第几行、哪个字段、当前值、期望格式 |
| 关键操作反馈 | 测试连接、导出、上传校验、写入、回退均需结果反馈 |
| 导入不直接写库 | 必须经过校验 + 差异预览 |
| 结构变化优先阻断 | 不一致则禁止写入 |
