## P0 - 项目骨架 + 基础管理

| # | 任务 | 类型 | 说明 |
|---|---|---|---|
| P0-1 | 前端项目初始化 | 前端 | Vite + React + Ant Design + React Router，跑通基础页面框架 |
| P0-2 | 后端项目初始化 | 后端 | FastAPI 项目结构、ORM 配置、数据库迁移脚本 |
| P0-3 | 平台数据库建表 | 后端 | 8 张核心表 DDL，SQLite 优先 |
| P0-4 | 文件夹部署结构 + 启动脚本 | 工程 | 搭建 `data-ops-workbench/` 目录结构，编写 start.sh / start.bat，FastAPI 托管前端静态资源 |
| P0-5 | 数据源管理-后端 | 后端 | CRUD 接口 + 测试连接接口（MySQL/PostgreSQL/SQL Server）|
| P0-6 | 数据源管理-前端 | 前端 | 列表页 + 新建/编辑页 + 测试连接交互 |

**P0 完成标志：** 能新建一个 MySQL 数据源并测试连接成功。

---

## P1 - 配置链路

| # | 任务 | 类型 | 说明 |
|---|---|---|---|
| P1-1 | 纳管表配置-后端 | 后端 | 获取库表清单、新建纳管表、保存配置、表结构检测接口 |
| P1-2 | 纳管表配置-前端 | 前端 | 纳管表列表页 + 表配置详情页 |
| P1-3 | 字段自动拉取-后端 | 后端 | 读取字段名/类型/样例数据，生成默认字段配置 |
| P1-4 | 字段配置-前端 | 前端 | 字段配置页，支持逐字段编辑、批量操作 |
| P1-5 | 字段配置-后端 | 后端 | 字段配置 CRUD 接口 |
| P1-6 | 表结构变化检测 | 后端 | 进入表或导入前自动比对 hash，结构不一致则阻断 |

**P1 完成标志：** 能选表、拉字段、配置编辑规则、检测结构变化。

---

## P2 - 核心操作链路

| # | 任务 | 类型 | 说明 |
|---|---|---|---|
| P2-1 | 数据浏览-后端 | 后端 | 分页读取业务表数据、关键字筛选、字段筛选 |
| P2-2 | 数据浏览-前端 | 前端 | 数据维护列表页 + 数据浏览页（动态表格、分页、横向滚动）|
| P2-3 | 模板导出-后端 | 后端 | 生成含隐藏元信息的 Excel 模板，记录导出日志 |
| P2-4 | 模板导出-前端 | 前端 | 导出确认弹窗 + 下载 |
| P2-5 | 模板导入-后端 | 后端 | 解析上传模板、校验合法性/版本/字段/数据类型 |
| P2-6 | 模板导入-前端 | 前端 | 模板导入页（拖拽上传 + 文件信息展示）|
| P2-7 | 导入校验结果-前端 | 前端 | 校验结果页（汇总 + 明细表格 + 导出错误明细）|
| P2-8 | 差异预览-后端 | 后端 | 原值/新值对比，生成差异数据 |
| P2-9 | 差异预览-前端 | 前端 | 差异预览页（差异表格 + 汇总卡片）|
| P2-10 | 安全回写-后端 | 后端 | 写前全表备份 + 执行 UPDATE + 记录回写日志 + 生成备份版本 |
| P2-11 | 写入结果反馈-前端 | 前端 | 写入结果弹窗/页面 |

**P2 完成标志：** 主链路完整走通——从浏览到导出到修改到导入校验到差异预览到回写。
