## 四、发现的 Bug 及修复

### Bug #1：数据源类型校验不完整 — 🔧 已修复
- **位置**: `backend/app/schemas/datasource.py`
- **问题**: db_type 正则只允许 `mysql|postgresql|sqlserver`，缺少 `oracle|dm|kingbase|sqlite`
- **修复**: 扩展正则为 `^(mysql|postgresql|sqlserver|oracle|dm|kingbase|sqlite)$`

### Bug #2：SQL Server list_tables 报 ODBC SQL type -150 错误 — 🔧 已修复
- **位置**: `backend/app/utils/remote_db.py` list_tables 函数
- **问题**: `sys.extended_properties.value` 是 `sql_variant` 类型，pyodbc 不支持
- **修复**: 将 `ep.value` 改为 `CAST(ep.value AS NVARCHAR(MAX))`

### Bug #3：SQLite 密码字段校验不兼容 — 🔧 已修复
- **位置**: `backend/app/schemas/datasource.py`
- **问题**: password 字段 `min_length=1` 导致 SQLite（无密码）创建失败
- **修复**: 改为 `min_length=0`
