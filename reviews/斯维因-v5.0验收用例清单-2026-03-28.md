# 斯维因 — v5.0 验收用例清单

> 整理人：斯维因  
> 日期：2026-03-28

---

## 一、用例范围

本用例清单用于验证 v5.0 统一认证接入是否达到发布准备条件。

---

## 二、核心用例

### UC-01 未登录进入工作台
**前置条件**：本地无 token  
**步骤**：打开工作台  
**期望结果**：显示统一认证登录页

### UC-02 发起微信登录
**前置条件**：处于登录页  
**步骤**：点击“微信登录”  
**期望结果**：跳转到统一认证平台微信授权地址

### UC-03 登录成功回跳
**前置条件**：微信授权成功  
**步骤**：完成授权  
**期望结果**：工作台回调页成功接收 token 并进入首页

### UC-04 token 在线验证成功
**前置条件**：已获得有效 token  
**步骤**：工作台调用 verify 接口  
**期望结果**：返回 valid=true，页面进入已登录状态

### UC-05 token 无效
**前置条件**：本地 token 无效  
**步骤**：进入工作台  
**期望结果**：提示登录失效并要求重新登录

### UC-06 token 过期
**前置条件**：本地 token 已过期  
**步骤**：进入工作台或刷新页面  
**期望结果**：提示登录已过期并返回登录页

### UC-07 已授权插件访问
**前置条件**：账号对目标插件已授权  
**步骤**：进入目标插件功能页  
**期望结果**：正常访问

### UC-08 未授权插件访问
**前置条件**：账号对目标插件未授权  
**步骤**：进入目标插件功能页  
**期望结果**：显示未授权提示，不允许进入功能主体

### UC-09 公钥下载成功
**前置条件**：认证平台在线  
**步骤**：工作台请求下载公钥  
**期望结果**：成功获取 PEM 格式公钥

### UC-10 本地离线验签成功
**前置条件**：已获取公钥，token 签名正确  
**步骤**：进行本地 RS256 验签  
**期望结果**：验签通过

### UC-11 Linux 打包后登录验证
**前置条件**：Linux 包构建完成  
**步骤**：启动应用并执行登录流程  
**期望结果**：应用可正常启动、登录、进入首页

### UC-12 Windows 打包后登录验证
**前置条件**：Windows 包构建完成  
**步骤**：启动应用并执行登录流程  
**期望结果**：应用可正常启动、登录、进入首页

## 三、补充用例

### UC-13 verify 接口异常
**期望结果**：提示认证服务异常，不允许静默通过

### UC-14 license 接口异常
**期望结果**：插件授权状态不可判定时，有明确提示，不允许误放行

### UC-15 公钥下载失败
**期望结果**：提示本地校验不可用

### UC-16 页面刷新保持登录态
**期望结果**：在 token 有效前提下，刷新不丢登录态

### UC-17 退出登录
**期望结果**：清理 token 与授权缓存，返回登录页

---

## 四、验收结论口径

若以下条件同时成立，则可判定 v5.0 达到发布准备：

1. 核心用例全部通过
2. 补充用例不存在 P0 / P1 缺陷
3. 双平台打包可运行
4. 登录、授权、验签三个闭环均成立
