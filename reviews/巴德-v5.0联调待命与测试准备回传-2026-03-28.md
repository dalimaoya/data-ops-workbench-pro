## 四、测试范围摘要

本轮待启动测试范围摘要如下：

### A. 登录与登录态
- 未登录进入工作台是否跳转登录页
- 微信登录跳转是否正确
- 回调后 token 是否成功接收并写入
- 登录成功是否进入首页
- 刷新后登录态是否保留
- 退出登录是否清理 token 与缓存

### B. token 校验与异常链路
- 有效 token 在线校验是否通过
- 无效 token 是否被识别并要求重登
- 过期 token 是否被识别并返回登录页
- verify 接口异常时是否有明确提示，且不静默放行

### C. 插件授权链路
- 至少 1 个插件已接入授权校验
- 已授权账号是否可进入
- 未授权账号是否被正确拦截
- 提示文案是否明确
- license 接口异常时是否避免误放行

### D. 公钥与离线验签
- 公钥是否可下载成功
- 是否能本地缓存或持久化
- 是否可基于本地公钥完成 RS256 验签
- 公钥下载失败时是否有提示
- 在线 verify 不可用时是否有明确处理策略

### E. 双平台冒烟
- Linux 包启动、登录、授权功能访问、未授权提示、退出登录
- Windows 包启动、登录、授权功能访问、未授权提示、退出登录
