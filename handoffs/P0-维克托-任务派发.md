## 重要：产品形态
交付物是一个完整的文件夹，不是 Tauri/Electron 应用：
- 文件夹拷贝到 Linux 或 Windows 即可运行
- Linux 运行 `./start.sh`，Windows 双击 `start.bat`
- 启动本地 Web 服务，用户通过浏览器访问
- 后端 FastAPI 同时托管前端静态资源，单进程
- 用户不需要安装任何额外环境
