# 测试任务：v3.5 模板安全加固

## 阶段
迭代测试补测（第一批 P0 第 3 项）

## 背景
v3.5 新增模板 Excel 保护和导入校验增强（commit 1c5077c），是数据安全基线功能。未经巴德正式测试。

## 测试范围
基于 CHANGELOG v3.5.0 功能点：

### 模板 Excel 保护
1. 表头行锁定（不可修改/删除）
2. 列结构保护（不可删除列或插入新列）
3. 主键列已有数据行只读
4. 工作表保护（带密码）
5. 只读单元格浅灰背景，可编辑单元格白色，底部预留新增数据区浅黄提示

### 导入校验增强
6. 列数不匹配阻断（提示具体差异）
7. 列名不匹配阻断（提示具体位置）
8. 主键不可变校验（提示具体行和原值/新值）
9. 日期类型校验
10. 所有校验错误精确到行号 + 列名

### 测试要点
- [ ] 导出模板后，用 Excel 打开验证表头行不可编辑
- [ ] 验证不可删除列、不可插入新列
- [ ] 主键列已有数据不可修改
- [ ] 工作表保护已启用
- [ ] 单元格颜色区分（灰/白/黄）
- [ ] 篡改列数后导入 → 阻断并提示具体差异
- [ ] 篡改列名后导入 → 阻断并提示具体位置
- [ ] 篡改主键值后导入 → 校验报错并提示行号和原值/新值
- [ ] 日期字段填写非法日期 → 校验报错
- [ ] 校验错误信息包含行号和列名
