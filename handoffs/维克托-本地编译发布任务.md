## 编译步骤

### Linux 编译
1. 确保 Python 虚拟环境和依赖就绪
2. 安装 Nuitka（如未安装）
3. 参考 build.sh 执行 Nuitka standalone 编译
4. 产物打包为 tar.gz，放入 release/ 目录

### Windows 编译
1. SSH 连接 Windows 服务器
2. 克隆或同步项目代码
3. 确保 Python 环境和依赖就绪
4. 安装 Nuitka（如未安装）
5. 执行 Nuitka standalone 编译
6. 产物打包为 zip
7. 拷贝回本机 release/ 目录

### 发布包组装
- Linux: `data-ops-workbench-v4.5.0-linux-x64.tar.gz`
- Windows: `data-ops-workbench-v4.5.0-windows-x64.zip`
- 两个包都需包含：编译产物 + 前端静态资源 + 启动脚本
