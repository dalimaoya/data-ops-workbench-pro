## 背景
v3.10 新增了 DELETE 回写支持，由维克托开发但未完整交付，部分代码由产品侧补充提交（commit dbe6d7a）。此版本未经巴德正式测试，现需补测。

## 测试范围
基于 CHANGELOG v3.10.0 的功能点：

### 功能点
1. **操作列**：数据浏览页新增操作列，支持标记行为「待删除」
2. **导入校验**：导入校验自动检测删除标记
3. **差异预览**：差异预览区分更新行（蓝色）、新增行（绿色）、删除行（红色）
4. **DELETE 执行**：回写执行 DELETE（先备份），结果页显示删除数

### 测试要点
- [ ] 标记待删除 → 导出 → 导入 → 校验识别删除标记
- [ ] 差异预览正确区分三种操作类型的颜色
- [ ] DELETE 执行前自动备份
- [ ] DELETE 执行后数据确实从业务表删除
- [ ] 结果页正确显示删除行数
- [ ] 混合场景：同一批次含更新 + 新增 + 删除
- [ ] 边界：标记删除后取消标记
- [ ] 边界：删除不存在的行（主键已被删除）
- [ ] 权限：只读用户不可标记删除
- [ ] 回退：删除后可通过版本回退恢复
