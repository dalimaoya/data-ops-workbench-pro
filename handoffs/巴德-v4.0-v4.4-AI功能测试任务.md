# 测试任务：v4.0 ~ v4.4 AI 功能全量测试

## 阶段
迭代测试补测（第二批 P1）

## 背景
v4.0 到 v4.4 共 5 个版本，实现了 AI 基础设施和 5 个 AI 功能场景。全部未经巴德正式测试。因功能关联度高（共享 AI 基础设施），合并为一个测试任务。

## 版本与功能清单

### v4.0 — AI 智能字段配置 + AI 基础设施
- AI 设置页：9 个大模型平台预设、OpenAI + Claude 双协议
- AIEngine 统一调度：规则引擎 → 统计分析 → LLM 适配器
- 置信度阈值体系：≥0.8 直接展示、0.5-0.8 标黄、<0.5 不展示
- 7 个 AI 功能独立开关
- 字段配置页「AI 智能推荐」：语义名/只读/枚举值推荐

### v4.1 — AI 导入数据智能校验
- 四类检测：异常值、格式异常、重复值、跨字段逻辑冲突
- 规则引擎兜底 + LLM 增强

### v4.2 — AI 自然语言数据查询
- 自然语言转筛选条件
- 规则引擎关键词匹配 + LLM 复杂语义

### v4.3 — AI 操作日志智能分析
- 操作频次统计、异常模式检测、风险评估
- LLM 增强自然语言摘要

### v4.4 — AI 批量智能填充
- 四层模式检测：数字递增、日期递增、关联填充、频率填充
- 置信度评分 + LLM 增强

## 测试要点

### AI 基础设施（v4.0）
- [ ] AI 设置页可配置模型平台和 API Key
- [ ] AI 功能开关独立控制（关闭某功能后对应入口隐藏）
- [ ] AIEngine 在无 LLM 配置时降级到规则引擎（不报错）
- [ ] 置信度阈值体系：≥0.8 直接展示、0.5-0.8 标黄、<0.5 过滤

### AI 智能字段配置（v4.0）
- [ ] 字段配置页有「AI 智能推荐」按钮
- [ ] 推荐结果含语义名、只读建议、枚举值
- [ ] 支持逐条采纳/全部采纳/仅采纳语义名

### AI 导入智能校验（v4.1）
- [ ] 导入校验结果页有「AI 智能校验」按钮
- [ ] 四类检测（异常值/格式/重复/跨字段）能正确识别
- [ ] 校验结果含严重等级和修复建议

### AI 自然语言查询（v4.2）
- [ ] 数据浏览页有「AI 查询」按钮
- [ ] 自然语言输入后生成筛选条件预览
- [ ] 筛选条件可一键应用到数据浏览

### AI 日志智能分析（v4.3）
- [ ] 日志中心有「AI 分析」按钮
- [ ] 分析结果含操作摘要、异常检测、风险评估
- [ ] 支持时间范围选择

### AI 批量智能填充（v4.4）
- [ ] 数据浏览页有「AI 智能填充」按钮
- [ ] 三步交互：选列 → 检测模式 → 预览建议值
- [ ] 填充值紫色高亮，置信度可视化
- [ ] 确认后进入标准差异预览回写流程
