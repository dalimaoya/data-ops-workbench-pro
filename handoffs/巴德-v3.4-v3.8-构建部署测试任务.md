## 版本与功能清单

### v3.4 — Nuitka 编译 + CI 自动发布

- 双平台支持（Linux x64 tar.gz + Windows x64 zip）
- GitHub Actions CI：推送 v* tag 自动触发编译 + 上传 Releases
- 仓库拆分：公开仓库（10 插件）+ 私有仓库（19 插件 + AI）

### v3.4.2 — Windows 三项修复
- 启动器 exe 无反应 → VBS 备选方案 + CI 降级
- 服务启动慢 → 后台启动 + 端口检测 + 健康检查轮询
- 自动打开浏览器

### v3.7 — Inno Setup 安装器
- Inno Setup 脚本生成 Windows .exe 安装程序
- 安装器自动检测/创建桌面快捷方式
- CI 集成：tag 推送自动触发编译 + 生成安装器 + 上传 Releases

### v3.8 — 启动器版本更新检查

- 有新版本弹窗提示，可跳转下载
