- 问题：时间格式显示异常

- 位置：数据维护 → 数据表维护 → 导入校验结果 → 差异预览
- 问题：从差异预览点"查看差异预览"或"返回校验结果"后直接回到上传页，而非校验结果页
- 修复：返回按钮路由目标改为校验结果页，保持导入任务 ID 上下文

### BUG-03：库表管理选数据源后显示 No data
- 位置：数据维护 → 库表管理 → 选择数据源
- 问题：选中数据源后不加载表列表
- 修复：修复数据加载逻辑，同时默认选中第 1 个数据源

### BUG-04：SQL 控制台选数据源后显示 No data
- 位置：数据维护 → SQL 控制台
- 问题：同 BUG-03
- 修复：修复数据加载逻辑，同时默认选中第 1 个数据源

### BUG-05：数据库维护分页选择器失效 + 滚动条
- 位置：数据维护 → 数据库维护
- 问题：分页 20/page 选择器切换无效，且内容被框体约束需要滚动条
- 修复：
  - 修复分页组件的 onChange/onShowSizeChange 绑定
  - 表格区域去掉固定高度/滚动条，改为全页面展示所有行

### BUG-06：导入数据返回后不回到原数据源页面
- 位置：数据维护 → 数据库维护 → 点击"导入数据" → 模板导入页 → 返回
- 问题：返回后回到数据库维护默认页，丢失了已选数据源状态
- 修复：返回时通过路由参数或状态保持已选数据源 ID

### BUG-07：插件中心每个插件点不开
- 位置：插件中心
- 问题：点击插件卡片无响应
- 修复：检查是否缺少点击事件或详情展开逻辑。如果当前设计不需要详情页，则点击卡片展开描述详情面板；如果只需要启用/停用，确保 Switch 可正常操作
