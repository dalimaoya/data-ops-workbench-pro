# 巴德 — v4.9 Windows 兼容性测试任务

> 派发人：斯维因 | 日期：2026-03-27 | 优先级：P1

---

## 背景

v4.9 引入了 Windows GUI 启动器（tkinter + pystray + psutil），需要在主流 Windows 版本上做兼容性验证，确认无启动异常、依赖缺失或 UI 渲染问题。

---

## 测试范围

**目标系统：**
- Windows 10（最新补丁版本）
- Windows 11（最新补丁版本）

**不在范围内：** Win7 / Win8 / Win8.1（已停止支持，不投入测试资源）

---

## 测试方式

以**静态代码审查 + 依赖兼容性分析**为主（无需真实 Windows 环境）：

1. 检查 `launcher.py` 使用的所有依赖库（tkinter、pystray、psutil、webbrowser、subprocess 等）在 Win10 / Win11 下的兼容性
2. 检查 `setup.iss`（Inno Setup 安装器）配置，确认目标平台声明正确
3. 检查 `build.bat` / `start.bat` 脚本在 Win10 / Win11 CMD / PowerShell 下的语法兼容性
4. 检查 `os.path`、文件路径分隔符等是否存在跨平台隐患
5. 检查 `CREATE_NO_WINDOW` 等 Win32 API 调用在 Win10 / Win11 下的可用性

---

## 重点检查项

| # | 检查点 | 说明 |
|---|--------|------|
| C1 | tkinter 可用性 | Win10/11 Python 3.x 默认包含 tkinter，确认无需额外安装 |
| C2 | pystray 兼容性 | 托盘功能在 Win10/11 下的已知问题 |
| C3 | psutil Win10/11 | CPU/内存读取接口兼容性 |
| C4 | Nuitka 编译产物 | 编译后 .exe 在 Win10/11 无需 Python 运行时的依赖完整性 |
| C5 | 安装器 UAC 提权 | setup.iss 中是否有正确的权限声明（PrivilegesRequired） |
| C6 | 端口占用检测 | `netstat + taskkill` 命令在 Win10/11 的可用性 |
| C7 | 防火墙 / 杀毒软件 | GUI 启动器是否可能被 Win11 Defender 拦截（常见误报场景分析） |
| C8 | 高 DPI 屏幕适配 | tkinter 在 Win11 高分辨率屏幕下是否有模糊/缩放问题 |
