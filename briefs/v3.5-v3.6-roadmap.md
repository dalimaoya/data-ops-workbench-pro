# 3.x 剩余版本路线图

## v3.4 — Windows 版本 + CI 自动发布
- GitHub Actions 配置（Linux + Windows 双平台自动编译）
- Windows Nuitka 编译包
- 打 tag 自动发布到 GitHub Releases
- 状态：进行中

## v3.5 — 模板安全加固
- 表头行保护（锁定不可修改/删除）
- 列结构保护（不允许删除列）
- 主键列保护（已有数据行主键只读）
- 工作表保护（Excel 保护功能，仅允许编辑可编辑字段）
- 导入校验增强：列数校验、列名校验、主键不可变校验
- 新增行预留空间（为 v4.0 INSERT 能力提前兼容）

## v3.6 — 用户权限管理
- 用户列表/新增/编辑/禁用/重置密码
- 个人设置（修改密码、修改显示名）
- 角色权限（管理员/操作员/只读）
- 前端权限隔离（不同角色看到不同菜单和按钮）
- 后端权限校验（API 层面拦截越权操作）
- 操作日志记录实际操作人

## v4.0 — AI 增强（已有完整设计文档）
- 智能字段配置助手
- 导入数据智能校验
- 自然语言查询
- 日志智能分析
- 批量智能填充
- 新增/删除行回写
