# v3.7 — Inno Setup Windows 安装包

## 目标
把 Windows 版本打成标准 .exe 安装包，双击安装，自动创建桌面快捷方式和开始菜单项。

## 功能
1. 用 Inno Setup 打包，产出 `data-ops-workbench-v3.7.0-win-x64-setup.exe`
2. 安装过程：
   - 选择安装目录（默认 C:\Program Files\数据运维工作台）
   - 创建桌面快捷方式（指向「数据运维工作台.exe」启动器）
   - 创建开始菜单文件夹
   - 可选创建开机自启（注册表或启动文件夹）
3. 卸载支持：控制面板 → 卸载程序
4. 安装时自动创建 data/backups/logs 目录
5. 安装包图标使用项目 logo

## CI 集成
在 GitHub Actions Windows job 中：
1. 下载 Inno Setup CLI（iscc.exe）
2. 编写 setup.iss 脚本
3. 编译产出 setup.exe
4. 上传到 Release（和 zip 包并列）

## 验收标准
1. ✅ 双击 setup.exe 能正常安装
2. ✅ 桌面出现快捷方式
3. ✅ 双击快捷方式启动，服务正常运行
4. ✅ 控制面板能正常卸载
5. ✅ CI 自动产出 setup.exe 并上传到 Release
