> 整理人：斯维因  
> 日期：2026-03-28

---

## 一、目标

为数据运维工作台补充统一认证接入后的页面结构与交互口径，供维克托直接实现、供巴德后续测试。

本说明聚焦：
1. 登录入口页
2. 回调处理中间页
3. 登录后状态展示
4. 插件未授权提示
5. 异常场景提示

---

## 二、页面清单

### 1. 登录页（新增）
用途：未登录用户进入工作台时的统一认证入口。

#### 页面元素
- 产品名称 / Logo
- 一句说明文案：通过统一认证平台完成身份登录
- 主按钮：`微信登录`
- 次要说明：登录后将自动返回工作台
- 底部状态区：
  - 服务正常 / 服务异常提示
  - 最近错误提示（如有）

#### 交互
- 点击“微信登录”后，跳转到统一认证平台微信登录地址
- 如本地已存在有效 token，则不展示登录页，直接进入首页

---

### 2. 登录回调页（新增）
用途：处理统一认证平台回跳后的 token 接收与校验。

#### 页面状态
1. `正在接收登录信息...`
2. `正在校验身份...`
3. `登录成功，正在进入工作台...`
4. `登录失败，请重新登录`

#### 处理逻辑
- 从 URL fragment 中提取 token
- 写入本地安全存储
- 调用 `/api/auth/verify`
- 成功则跳首页
- 失败则清理 token 并返回登录页

---

### 3. 登录后用户状态区（新增或补充）
建议位置：右上角用户区 / 设置页账号区。

#### 展示内容
- 登录状态：已登录 / 未登录
- `account_id`
- token 到期时间
- 认证来源：统一认证平台
- 操作：`退出登录`

#### 退出登录行为
- 清理本地 token
- 清理授权缓存
- 返回登录页

---

### 4. 插件未授权提示页 / 提示区（新增）
用途：当用户进入受控插件功能但当前账号未授权时显示。

#### 提示内容
- 标题：`当前账号未开通此功能`
- 正文：你当前账号暂无该插件使用权限，请联系管理员或更换授权账号
- 如接口返回到期时间，可展示：
  - 授权到期时间
  - 当前插件标识

#### 交互
- 返回首页
- 返回上一页
- 保持页面明确，不允许空白页或静默失败

---

## 三、异常场景提示要求

### 1. 无 token
提示：`当前未登录，请先完成微信登录`

### 2. token 无效 / 已过期
提示：`登录状态已失效，请重新登录`

### 3. 认证平台不可达
提示：`认证服务暂时不可用，请稍后重试`

### 4. 在线验证失败但本地离线验签通过
实现上允许保留受控态，但页面应可提示：
`当前处于离线校验状态，部分在线能力可能受限`

### 5. 公钥下载失败
提示：`公钥获取失败，暂无法完成本地校验`

---

## 四、推荐页面跳转关系

### 未登录
登录页 → 微信授权跳转 → 回调页 → 首页

### 已登录
首页 → 受控插件页 → 授权校验 →
- 已授权：正常进入
- 未授权：未授权提示页/提示区

### 登录失效
任意页 → 检测到 token 失效 → 登录页

---

## 五、实现建议

1. 登录页和回调页应尽量轻量，不引入复杂逻辑
2. token 管理统一封装，不允许页面散落处理
3. 授权校验应有统一封装，避免每个页面各自拼接
4. 错误提示需可复用，避免页面之间文案不一致
5. Linux / Windows 打包后页面行为必须保持一致

---
