## 一、版本编号结论

经李琪确认，本次统一认证接入版本正式编号定为：

> **v5.0**

---

## 二、版本命名理由

本次变更已超出普通修复或局部增强范围，属于版本级能力升级，原因包括：

1. 登录方式升级
   - 工作台接入统一认证平台
   - 用户登录路径发生结构性变化

2. 权限模型升级
   - 新增 token 在线校验
   - 新增插件授权校验
   - 新增公钥下载与本地离线验签

3. 架构边界变化
   - 工作台开始与外部统一认证平台联动
   - 引入新的认证与授权依赖关系

4. 发布要求升级
   - 需要完成 Linux / Windows 双平台编译打包验证

因此不再沿用 v4.x 的修补性版本语义，而采用新的主版本编号。

## 三、v5.0 范围定义

### 本期纳入范围
1. 微信公众号统一认证登录接入
2. token 在线校验
3. 插件授权校验
4. 公钥下载与本地离线验签
5. 统一认证接入后的联调验证
6. Linux / Windows 双平台打包验证

### 暂不纳入 v5.0 首批范围
1. 多账号切换
2. 多组织切换
3. 认证平台管理后台嵌入工作台
4. 激活码管理后台嵌入工作台
5. 复杂 SSO 配置中心
