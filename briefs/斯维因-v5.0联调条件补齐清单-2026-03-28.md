## 二、当前仍缺的联调条件

### 1. 有效测试账号
需要至少 1 个可用于真实微信登录 / 认证联调的测试账号。

### 2. 有效 token 或可触发真实登录的联调条件
需要至少具备其一：
- 可直接用于验证的有效 token
- 或能完成真实登录的测试流程

### 3. 回调地址白名单
需要确认当前工作台回调地址已被统一认证平台允许回跳，否则登录成功闭环无法成立。

### 4. v5.0 双平台产物
需要至少具备：
- Linux v5.0 可测产物
- Windows v5.0 可测产物

否则双平台最小冒烟无法成立。

---

## 三、补齐后的复测顺序

在上述条件补齐后，按以下顺序推进：

1. 真实登录成功闭环
2. token verify 正向验证
3. 插件授权放行 / 拦截验证
4. 公钥下载 + 首次离线验签验证
5. Linux 最小冒烟
6. Windows 最小冒烟
