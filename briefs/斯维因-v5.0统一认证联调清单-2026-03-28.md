## 三、联调清单

### A. 登录闭环
- [ ] 登录页展示正常
- [ ] 点击“微信登录”可正确跳转认证平台
- [ ] 微信授权完成后可回跳工作台
- [ ] 回调页可正确提取 token
- [ ] token 写入本地成功
- [ ] 调用 `/api/auth/verify` 成功
- [ ] 登录成功后进入首页
- [ ] 刷新后登录态可恢复

### B. token 异常链路
- [ ] 无 token 时正确跳登录页
- [ ] token 无效时提示重新登录
- [ ] token 过期时提示重新登录
- [ ] verify 接口异常时有明确提示

### C. 插件授权闭环
- [ ] 至少 1 个插件功能已接入授权校验
- [ ] 已授权账号可进入功能
- [ ] 未授权账号被正确拦截
- [ ] 未授权提示文案清晰
- [ ] 授权结果无误放行 / 误拦截

### D. 公钥与离线验签闭环
- [ ] 工作台可成功下载公钥
- [ ] 公钥可本地缓存或持久化
- [ ] JWT 可通过本地公钥完成 RS256 验签
- [ ] 在线 verify 不可用时有明确策略
- [ ] 离线验签状态下页面提示清晰（如已实现）

### E. 双平台运行闭环
- [ ] Linux 包可启动
- [ ] Linux 包可登录
- [ ] Linux 包可进入至少 1 个授权功能
- [ ] Windows 包可启动
- [ ] Windows 包可登录
- [ ] Windows 包可进入至少 1 个授权功能
