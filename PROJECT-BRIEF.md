# 数据运维工作台 — 项目摘要

> 提莫队长巡检用。最后更新：2026-03-28

## 一句话定义
为交付和运维现场提供"不碰数据库客户端也能安全修订结果表"的可视化工作台。

## 技术栈
- 后端：FastAPI + SQLAlchemy + SQLite（本地）
- 前端：React 19 + Ant Design 6 + TypeScript + Vite
- 支持 7 种数据库：MySQL/PostgreSQL/SQL Server/Oracle/达梦/人大金仓/SQLite
