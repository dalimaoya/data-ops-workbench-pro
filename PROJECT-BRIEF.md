# 数据运维工作台 — 项目摘要

> 最后更新：2026-04-04

## 一句话定义
为交付和运维现场提供"不碰数据库客户端也能安全修订结果表"的可视化工作台。

## 技术栈
- 后端：Python 3.11 + FastAPI + SQLAlchemy 2.0 + SQLite（平台DB）
- 前端：React 19 + Ant Design 6 + TypeScript + Vite + Recharts
- 认证：JWT + bcrypt + 统一认证平台（微信扫码 + 离线密码双模式）
- 打包：PyInstaller --onedir，GitHub Actions CI/CD
- 支持 7 种数据库：MySQL / PostgreSQL / SQL Server / Oracle / 达梦 / 人大金仓 / SQLite

## 当前状态
- **版本**：v5.4.2（2026-04-02 发布）
- **阶段**：功能稳定，持续迭代优化
- **已完成**：MVP + 统一认证 + 四级角色 + 18 插件体系 + 数据格式自适应 + SQL 操作台 + 备份迁移 + 批量操作 + 文档更新
- **开源状态**：公开仓库（AGPL-3.0）+ 私有仓库（Pro 版）

## 代码规模
